git commands:

git init : to initialize the git local repository.
git remote add origin "https://github.com/kalagonda/learning.git" : to configure the cloud repository with local repository
git pull origin master : to fetch all the files from cloud master repository.
git add file name : to add the file to local repository
git commit -m "description" : to save the changes in the local repository
git add -A : to add all the files to the local repository
git commit -a -m : to commit/save all the local repository files
git log : tracking  committed files
git branch branchname : to create a new branch
git checkout name_of_the_branch : to switch to specified branch
git status : to see the untracked/modified uncommitted files.
git merge branch_name : merges to the checked out branch to branch_name
git rebase branch_name : will to the same job of branch merge
ssh-keygen : to generate the ssh key for cloud repository connection to our account
ssh -T git@github.com : to authenticate ssh key with cloud repository
git push origin : to push to the cloud repository
