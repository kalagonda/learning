git commands: git init : to initialize the git local repository. git remote add origin "https://github.com/kalagonda/learning.git" : to configure the cloud repository with local repository git pull origin master : to fetch all the files from cloud master repository. git add file name : to add the file to local repository git commit -m "description" : to save the changes in the local repository git add -A : to add all the files to the local repository git commit -a -m : to commit/save all the local repository files git log : tracking committed files
